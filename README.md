# mybugscan
fofa_api+bugscan插件扫描
#用途
主要用于找靶机地址，也可以用于批量测试
#结构
```
│  1.txt
│  run.py
│  
└─bugscan
    │  exp.py
    │  exp.pyc
    │  __init__.py
    │  __init__.pyc
    │  
    └─dummy
            common2.pyc
            common3.pyc
            __init__.py
            __init__.pyc
```
#设置
1.请把exp放在bugscan目录下

2.请设置run.py的fofa_api
